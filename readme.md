## About project
This project was done with framework laravel 5.4, reworked Sentinel addon(https://cartalyst.com/manual/sentinel/2.0), Eloquent ORM, custom localstorage basket, facebook API and google mail API, MySQL database, a lot of jquery scripts(validation, sliders etc.), AJAX and Eloquent filters, Webpack and uploaded to Heroku https://w-eshop.herokuapp.com/.
